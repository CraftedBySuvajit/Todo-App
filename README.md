## ✨ Features

- **🧠 AI Task Architect:** Let AI break down your complex goals into actionable sub-tasks instantly using the Gemini AI Engine.
- **⚡ Real-time Sync:** Powered by Firebase Firestore, ensuring your tasks are always up to date across all your devices.
- **🔐 Secure Authentication:** Seamless Google Sign-In and anonymous "Guest Mode" for rapid access.
- **🎨 Premium UI/UX:** A stunning, modern glassmorphic design featuring smooth micro-animations, drag-and-drop reordering, and beautiful gradients.
- **📄 PDF Export:** Instantly generate professional, styled PDF reports of your task details and completion status.
- **📱 Fully Responsive:** Carefully crafted to look perfect on both desktop and mobile screens.

## 🛠️ Technology Stack

- **Frontend:** React 19, TypeScript, Vite
- **Styling:** Tailwind CSS v4, Lucide React (Icons), Motion (Framer Motion)
- **Backend & Database:** Firebase (Authentication, Firestore)
- **AI Integration:** Google GenAI SDK

## 🚀 Getting Started Locally

Follow these steps to run the project on your local machine:

### 1. Prerequisites
Make sure you have **Node.js** installed on your machine.

### 2. Installation
Clone the repository and install the dependencies:
```bash
npm install
```

### 3. Environment Setup
Create a `.env` file in the root of your project and configure your API keys. You can use `.env.example` as a template.

You will need:
- A Gemini API Key
- Your Firebase Project Configuration

```env
GEMINI_API_KEY="your_api_key_here"

VITE_FIREBASE_API_KEY="your_firebase_key"
VITE_FIREBASE_AUTH_DOMAIN="your_project.firebaseapp.com"
VITE_FIREBASE_PROJECT_ID="your_project_id"
VITE_FIREBASE_STORAGE_BUCKET="your_project.firebasestorage.app"
VITE_FIREBASE_MESSAGING_SENDER_ID="your_sender_id"
VITE_FIREBASE_APP_ID="your_app_id"
```

### 4. Run the Development Server
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:3000` to see the app in action!
